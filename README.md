# ESP-8266-ThingSpeak-Temp-Humidity-Monitor
A temperature and humidity sensor that sends data to ThingSpeak channel  

BOM:  
ESP-8266(ESP-01 version)  
AMS-1117 (5V to 3.3V step down module)
DHT-22 Temp/Humidity Sensor
